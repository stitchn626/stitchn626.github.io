Arduino

An electronic component.

• Arduino is an open-source platform used for building electronics projects.

• It is a development board developed by Arduino.cc and is based on Atmega328 Microcontroller.

• The software used for writing, compiling & uploading code to Arduino boards is called Arduino IDE (Integrated Development Environment), which is free to download from Arduino Official Site.

• It has an operating voltage of 5V while the input voltage may vary from 6V to 12V.

Arduino Structure

1) Power Source

The Arduino UNO can be powered from a USB cable coming from your computer or a wall power supply terminated in a barrel jack. The USB cable also helps to upload code into the board.

2) Pins

This includes 5V, 3.3V, GND, Analog, Digital, PWM, AREF, Vin pins. These pins are where you connect wires to build a circuit (typically through a breadboard).

  • GND: Ground pins.
  • 5V & 3.3V: Supplies 5 volts and 3.3 volts respectively.
  • Analog: A0–A5, used for reading analog signals.
  • Digital: 0–13, used for digital input/output.
  • PWM: Marked with ~ (pins 3, 5, 6, 9, 10, 11), used for analog output via PWM.
  • AREF: Analog Reference, used to set an external reference voltage.

3) Reset Button

Pushing this button restarts any code loaded on the Arduino.

4) Power LED Indicator

This LED lights up when the board is powered, indicating it's working correctly.

5) TX RX LEDs

TX (Transmit) and RX (Receive) LEDs blink during data transmission or receiving.

6) Voltage Regulator

Regulates the voltage coming into the Arduino to ensure safe operation (up to 20V).

Features

  • ATmega328P microcontroller (16 MHz)
  • Digital I/O pins with PWM capability
  • 10-bit Analog-to-Digital Converter (ADC)
  • Power via USB, external supply, or voltage regulator
  • Supports Serial (UART), I2C, and SPI communication
  • USB interface for programming and communication
  • Reset button
  • Onboard LEDs
  • Compatibility with shields and accessories

Applications

  • Prototyping and education
  • Home automation
  • Robotics
  • Internet of Things (IoT)
  • Wearable electronics
  • Embedded systems
  • Security systems
