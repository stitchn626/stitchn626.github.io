This project serves both as a learning record and a practical portfolio of code examples and report-writing tips.

Introduction

HTML stands for HyperText Markup Language. It is used to define the structure and layout of a webpage. With HTML, we tell the browser what content to display and how it should be organized.

While CSS styles the content and JavaScript adds behavior, HTML provides the skeleton.

Basic Structure of an HTML Document

<!DOCTYPE html>
<html>
  <head>
    <title>My First Webpage</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>My first web page created.</p>
  </body>
</html>

Explanation of Structure:

  • <!DOCTYPE html>: Declares the document as HTML5. It must be the first line of your HTML file.
  • <html>: Root element that wraps the entire content of your HTML page.
  • <head>: Contains metadata and information like:
    • <title>: Sets the page title in the browser tab
    • <meta>: Defines character set, viewport, etc.
    • <link>: Links to external stylesheets
  • <body>: Contains all visible content like text, images, and links. Everything that appears in the browser goes here.

Commonly Used HTML Tags

  1. Text Tags

    • <h1> to <h6>: Headings from largest (<h1>) to smallest (<h6>)
    • <p>: Paragraph tag
    • <br>: Line break
    • <hr>: Horizontal line

  2. Links and Images

    • <a href="URL">Link Text</a>: Anchor tag for hyperlinks
    • <img src="image.jpg" alt="Description">: To display images

  3. Lists

    • <ul>: Unordered list (bullets)
    • <ol>: Ordered list (numbers)
    • <li>: List item (used inside <ul> or <ol>)
      example:
<ul>
  <li>Apples</li>
  <li>Bananas</li>
</ul>

  4. Tables

    • <table>: Starts the table
    • <tr>: Table row
    • <th>: Table header
    • <td>: Table cell (data)

  5. Semantic HTML Tags

    These tags give meaning to the structure and help with accessibility and SEO:

    • <header>: Top section of the page
    • <nav>: Navigation links
    • <main>: Main content
    • <section>: Group of related content
    • <article>: Self-contained content
    • <footer>: Bottom section
